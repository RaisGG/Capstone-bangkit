About Edify Diabetics
Edify is your personal assistant who can provide knowledge about fruits or vegetables suitable for consumption for diabetics.
Edify can recognize the fruits that you will consume just by using your smartphone camera.

About this Project
This project uses the Kotlin programming language. with android studio as its IDE. in the project using retrofit as a library that is used to simplify the process of exchanging data between android applications and servers through the REST API. Besides that, webview is also used to display the diabetes.org website which will provide articles about diabetes.
